configuration-modules-grid
--------------------------

Description
~~~~~~~~~~~

These NCM modules configure services for the EGEE/EGI Grid initiatives.

An incomplete list of what you can expect with our current library of Grid-related NCM components:

 - WMS client configuration
 - YAIM configurations and executions
 - Basic DCache configurations

and more.
